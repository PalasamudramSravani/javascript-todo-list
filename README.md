#  Todo List App

A simple and interactive Todo List web application built using **HTML, CSS, and JavaScript**.


##  Features

* Add new tasks
* Delete tasks easily
* Dynamic UI updates using JavaScript
* Uses event delegation for better performance


##  Technologies Used

* HTML
* CSS
* JavaScript (DOM Manipulation)


## 📂 Project Structure
📁 todo-list-app
 ┣ 📄 index.html
 ┣ 📄 style.css
 ┗ 📄 app.js


##  How It Works

* User enters a task in the input field
* Clicks the **Add Task** button
* Task is added to the list with a delete button
* Clicking delete removes the task instantly


##  Learning Purpose

This project helps in understanding:

* DOM manipulation
* Event handling
* Event delegation
* Basic UI interactions
